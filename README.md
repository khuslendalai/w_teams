# W Teams

A cross-platform worship team management application built with Flutter and Firebase.

W Teams helps teams organize members, schedules, events, and setlists in a single platform with real-time cloud synchronization.

![Flutter](https://img.shields.io/badge/Flutter-3.x-blue)
![Firebase](https://img.shields.io/badge/Firebase-Backend-orange)

---

## Tech Stack

### Languages

* Dart

### Frameworks & Services

* Flutter
* Firebase Authentication
* Cloud Firestore

### Tools

* Android Studio
* Git & GitHub

---

## Features

* User authentication with Firebase
* Team creation and invite code system
* Real-time member management
* Event scheduling and organization
* Setlist creation and management
* Firestore cloud database integration
* Role-based team structure
* Cross-platform Flutter architecture

## Getting Started

### Prerequisites

Make sure you have installed:

* Flutter SDK
* Android Studio
* Firebase CLI
* Git
* Android SDK

Check Flutter installation:

```bash
flutter doctor
```

---

## Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/w_teams.git
```

### Open the project

```bash
cd w_teams
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```

---

## Firebase Configuration

This project uses Firebase Authentication and Cloud Firestore.

Required Firebase setup:

* Create a Firebase project
* Add Android application
* Download `google-services.json`
* Place it inside:

```text
android/app/
```

---

## Environment Setup

No `.env` file is currently required.

Firebase configuration is handled through:

* `google-services.json`
