# W Teams

W Teams is a cross-platform worship team management application built with Flutter and Firebase.

## Features

* Team creation and invite system
* Member management
* Event scheduling
* Setlist and song management
* Firebase Authentication
* Cloud Firestore integration
* Real-time data updates

## Tech Stack

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore

## Project Structure

```text
lib/
 ├── models/
 ├── screens/
 ├── services/
 ├── widgets/
 └── main.dart
```

## Database Collections

* users
* teams
* events
* setlists
* members
* invitations
